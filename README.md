**University Course Management InterfaceOverview**

A web application that allows a University Supervisor to manage the university's course catalog. Built with React, Vite, Tailwind CSS, and Axios.
Features
Secure login with JWT authentication, dashboard listing all courses, create/view/edit/delete courses, loading spinners, and success or error messages on every action.
Tech Stack
React, Vite, Tailwind CSS v4, Axios, React Router DOM.
Getting Started
Clone the repository, run npm install to install dependencies, then npm run dev to start the development server. 
Test Credentials
Email: admin@example.com — Password: adminpassword123
API
Consumes a RESTful backend API. Key endpoints: POST /auth/login, GET /courses, GET /courses/:id, POST /courses, PUT /courses/:id, DELETE /courses/:id.
Deployment
Deployed on Vercel. Live link: https://university-course-management-interf.vercel.app/
